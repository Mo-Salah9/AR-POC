# كشف النص بالكاميرا (Web OCR)

تطبيق ويب يفتح **الكاميرا تلقائيًا** عند فتح الصفحة، يمسح النص العربي، وعند رصد **الرياضيات** يفتح [Google](https://www.google.com).

## التشغيل السريع (Windows)

1. افتح مجلد المشروع في الطرفية:
   ```powershell
   cd C:\Users\USER\Desktop\web-ar-opencv
   npm install
   npm start
   ```
   أو انقر مرتين على **`start.bat`** (بعد `npm install` مرة واحدة).

2. افتح المتصفح: **http://localhost:3000**

3. اسمح باستخدام الكاميرا.

4. للاختبار: افتح **http://localhost:3000/test-card.html** على شاشة أخرى أو اطبع الكلمة **الرياضيات** ووجّه الكاميرا نحوها.

## المتطلبات

- Chrome أو Edge (مُوصى به)
- **localhost** أو HTTPS (الكاميرا لا تعمل مع `file://`)
- اتصال إنترنت أول مرة (تحميل نموذج العربية لـ Tesseract)

## إضافة كلمات وروابط

عدّل `js/config.js`:

```javascript
export const TARGETS = [
  {
    text: "الرياضيات",
    url: "https://www.google.com",
    buttonLabel: "فتح Google",
  },
  {
    text: "العلوم",
    url: "https://example.com/science",
    buttonLabel: "فتح العلوم",
  },
];
```

## هيكل المشروع

| ملف | الوظيفة |
|-----|---------|
| `index.html` | الواجهة الرئيسية |
| `js/app.js` | الكاميرا + معالجة الصورة + OCR |
| `js/config.js` | الكلمات والروابط والإعدادات |
| `test-card.html` | صفحة اختبار بكلمة كبيرة |
| `start.bat` | تشغيل سريع على Windows |

## آلية العمل

1. `getUserMedia` — تشغيل الكاميرا (خلفية إن وُجدت).
2. كل ~1.2 ثانية: لقطة + تحسين تباين + Tesseract (عربي).
3. عند تطابق **الرياضيات** مرتين متتاليتين → فتح الرابط (مع مهلة 8 ثوانٍ بين الفتحات).

## نصائح لتحسين الكشف

- إضاءة جيدة ونص واضح وكبير
- ثبّت الكاميرا ثانية أو ثنتين على الكلمة
- راقب سطر «النص المكتشف» لمعرفة ما يراه OCR
